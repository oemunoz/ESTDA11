# Timber

Set of basic examples and workshops programs around C++.

## Installation

Tested on gcc version 4.8.4 and libncurses5:5.9+20140118
(Yes the compiler from Mars.)

## Usage

g++ timber.cpp -o timber.bin -lpanel -lncurses -lmenu -lform
